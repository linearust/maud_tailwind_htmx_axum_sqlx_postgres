# Project Guidelines

Template repository for minimal web apps. Breaking changes acceptable.

---

## UI (Project-Specific)

No shadows, rounded corners, font weights, or decorative backgrounds.
Simple borders, single primary color. Hover for interaction, border change for focus.
Tight spacing, maximum density.

---

## Architecture Notes

- Handler modules: `handlers/forms/`, `handlers/actions/`, `handlers/pages/`
- All paths defined in `src/paths.rs`
- Constants in `src/constants.rs`
